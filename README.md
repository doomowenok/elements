### Unity Version:
    2022.3.44f1

### Dependencies:
    UniTask
    Dotween
    VContainer

### How to run:
    * Open "Boot" scene in Assets/Scenes
    * Click Play button

### Add new level:
    * Create new level config 
        (Right Click -> Create -> Configs -> Core -> Level Grid)
    * Fill it with data
    * Add to Game Config (Can find in Assets/Resources/Configs)
